# My Electron App

A beginner-friendly desktop calender application built using Electron, HTML, CSS, JavaScript, and Node.js.

## 🚀 Features

- Desktop application using Electron
- Simple and clean user interface
- Built with web technologies
- Beginner-friendly project structure

## 🛠️ Technologies Used

- Electron
- Node.js
- HTML
- CSS
- JavaScript

## 📂 Project Structure

my-electron-app/
│
├── main.js

└── README.md


▶️ Installation

Clone the repository:

git clone https://github.com/Nourinbushair/my-electron-app.git

Go to the project folder:

cd my-electron-app

Install dependencies:

npm install

Run the application:

npm start
📖 Learning Purpose

This project was created for learning Electron desktop application development and understanding how web technologies can be used to build cross-platform desktop apps.
